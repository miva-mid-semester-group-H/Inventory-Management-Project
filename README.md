# Inventory Management System

A web-based Inventory Management System built by a student team at Miva Open University as part of our mid-semester group project.

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS
- **Backend:** Python, Flask
- **Database:** MySQL

## 👥 Team Structure

| Role | Responsibility |
|------|---------------|
| Frontend | UI design, HTML & CSS |
| Backend | Flask routes, sessions, form handling |
| Database | MySQL schema, queries, integration |

## 🚀 Features

- User Signup with form validation
- User Login with session management
- Protected Dashboard (only accessible when logged in)
- Logout functionality
- Clean separation of frontend, backend, and database layers

## ⚙️ How to Run Locally

1. Clone the repository
```bash
git clone https://github.com/miva-mid-semester-group-H/Inventory-Management-Project.git
```

2. Navigate into the project folder
```bash
cd Inventory-Management-Project
```

3. Install Flask
```bash
pip install flask
```

4. Run the app
```bash
python app.py
```

5. Open your browser and visit
```
http://127.0.0.1:5000
```
